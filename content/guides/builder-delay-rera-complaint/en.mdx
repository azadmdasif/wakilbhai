## State RERA portals

| State | Portal |
| --- | --- |
| Maharashtra (MahaRERA) | [maharera.maharashtra.gov.in](https://maharera.maharashtra.gov.in) |
| Uttar Pradesh (UP RERA) | [up-rera.in](https://www.up-rera.in) |
| Karnataka | [rera.karnataka.gov.in](https://rera.karnataka.gov.in) |
| Delhi | [rera.delhi.gov.in](https://rera.delhi.gov.in) |
| Haryana (HRERA Gurugram / Panchkula) | [haryanarera.gov.in](https://haryanarera.gov.in) |
| Telangana | [rera.telangana.gov.in](https://rera.telangana.gov.in) |
| Tamil Nadu (TNRERA) | [rera.tn.gov.in](https://rera.tn.gov.in) |
| Gujarat (GujRERA) | [gujrera.gujarat.gov.in](https://gujrera.gujarat.gov.in) |
| West Bengal (WBRERA) | [rera.wb.gov.in](https://rera.wb.gov.in) |
| Madhya Pradesh | [rera.mp.gov.in](https://rera.mp.gov.in) |
